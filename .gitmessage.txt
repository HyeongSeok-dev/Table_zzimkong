# 예) feat : 로그인 기능 추가

타입 목록
# feat : 새로운 기능 추가
# fix : 버그 수정
# docs : 문서 수정
# ci :  CI 관련 설정 수정 //ex) 머지등을 할때 사용
# test : 테스트 코드 추가
# refact : 코드 리팩토링 //  ex) 변수 이름 변경
# style : 코드 의미에 영향을 주지 않는 변경사항 //주석 들여쓰기 등 
# chore : 빌드 부분 혹은 패키지 매니저 수정사항
